# digiid-poc-wallet

Digitaalinen henkilöllisyys -hankkeen lompakkosovellus POC.

## Käyttöönotto

Avaa projekti Android Studiossa (Arctic Fox tai uudempi) ja lisää projektin juureen generoituun local.properties -tiedostoon tarvittavat asetukset:
   
```properties
githubUsername=github_packages_username
githubToken=github_packages_access_token

walletBaseUrl=http://url/to/wallet/service/
```
